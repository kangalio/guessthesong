Changelog

This is a list of improvements, fixes, additions, and changed features


Version 0.5.18

11-Dec-2022 17:59 current

  • Added skip button for lobby leader

Version 0.5.17

23-May-2021 12:37

  • Bug fixes and improvements to game performance
  • Limited number of people per room to 15. This was required for the game to run at a reasonable speed.

Version 0.5.15

07-Apr-2021 16:46

  • Added Top Songs of 2020 theme.

Version 0.5.14.3

22-Jan-2021 11:20

  • Fixed more playback issues.

Version 0.5.14.1

17-Jan-2021 14:26

  • Fixed a playback issue on Firefox browser.

Version 0.5.14

16-Jan-2021 16:27

  • Changed default number of rounds to 9 (was 12) and the default round time to 75 (was 90).
  • Disabled the Christmas theme.
  • Added support for more then one session for accounts.
  • Added a feature to logout on all places.
  • Fixed an issue where it would say that a player joined a player became room owner.
  • Fixed a bug with audio playback preventing songs from being played.
  • Changed the point scoring system.
  • Fixed a bug that caused players to disconnect from the game unexpectedly.
  • Fixed a bug causing the players to not render correctly on small screens.
  • Fixed various other issues.

26-Dec-2020 11:17

  • Fixed a game breaking bug

Version 0.5.11

26-Dec-2020 10:56

  • Added "Best Pop Songs of All Time" theme
  • Fixed a bug where players would not be removed from the room.
  • Various fixes and improvements

Version 0.5.5

06-Dec-2020 22:25

  • Added two new themes: "Greatest TV Theme Songs" and "House Music".
  • Moved some tabs around in the navbar.
  • Renamed the 'server browser' to 'room browser'.
  • The title of the room browser indicates how many rooms there are.
  • The room browser more elegantly shows information to the user.
  • Added a passive round counter to the game room.
  • Added the rounds, and seconds to the game room tab title, as well as a "new round" and "game over" message.

Version 0.5.4

05-Dec-2020 15:33

  • Added a global leaderboard
  • Various fixes and improvements

Version 0.5

04-Nov-2020 22:07

Major changes:

  • Added nightmode.
  • Overhauled lobby. There is now a chat and the room owner can change the game settings.

Minor changes:

  • Fixed various bugs and issues.

Version 0.4.4

25-Oct-2020 13:35

Major changes:

  • Redesigned game room page to scale a lot better for mobile phones and larger screens.
  • The game no longer breaks when a phone is in horizontal mode.
  • Added new themes: Random songs, and Halloween.

Minor changes:

  • Fixed a bug when rejoining the game when the game ended which causes an error.
  • Wins, losses, and games played are now correctly counted in a players profile.
  • When you rejoin the game when it just ended the scoreboard will be displayed again.
  • Compressed a couple of images to increase load times on slower connections.
  • Fixed an error when entering invalid input on the join game page.
  • Fixed rooms sometimes not having an owner.
  • Fixed missing meta tags, and alt text.
  • Fixed various bugs and issues.

Version 0.4.3

17-Oct-2020 12:05

Major changes:

  • Added new visualizer.
  • Added feature to create an account.
  • New themes added: '70s Hits', 'Best Rap Songs Of All Time', and 'Meme Songs'.
  • Added buttons to share on social media.
  • Added a character limit to the chat.
  • Changed rooms codes to be only 4 digits long.
  • Support for more than 8 players added.

Minor changes:

  • Added glyphicons to navbar.
  • Rooms now have 'owners'. The owner of a room is the only one who can start the game.
  • Named GuessTheSong's mascot Rosie.
  • Rosie blinks when someone guesses correctly.
  • Themes are sorted a bit more logically.
  • Changed the lobby so that the room password field is only visible if it is set.
  • Added player background to in game player character
  • Increased the guessing threshold to 0.92 from 0.88
  • Fixed various bugs.

Version 0.3.5

19-Sep-2020 16:50

  • Added a public changelog.
  • Added option to filter out explicit songs in themed game mode.
  • Added advanced options tab when creating room.

  • Added option to customize how long a round takes.

  • Changed the avatar selection in create and join room pages.

  • Fixed various other issues.
